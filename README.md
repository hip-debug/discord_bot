# Discord Экономика Бот

Бот для Discord с полной системой экономики, включая банковскую систему, работы, зарплаты и недвижимость.

## 📋 Функционал

### 💰 Банковская система
- `/баланс` - Проверить свой баланс
- `/депозит <сумма>` - Внести деньги в банк
- `/снять <сумма>` - Снять деньги из банка
- `/передать <пользователь> <сумма>` - Перевести деньги другому пользователю
- `/ежедневно` - Получить ежедневную награду ($500)

### 💼 Работа и профессии
- `/работы` - Показать список доступных работ
- `/устроиться <ID>` - Устроиться на работу
- `/работать` - Поработать и получить зарплату
- `/моя_работа` - Показать текущую работу

### 🏠 Недвижимость
- `/недвижимость` - Показать доступную недвижимость
- `/мои_дома` - Показать вашу недвижимость
- `/купить_дом <ID>` - Купить недвижимость
- `/доход` - Получить ежедневный доход от недвижимости

## 🚀 Установка

### 1. Клонирование репозитория
```bash
git clone <repository_url>
cd <project_directory>
```

### 2. Установка зависимостей
```bash
pip install -r requirements.txt
```

### 3. Настройка переменных окружения
Скопируйте файл `.env.example` в `.env` и заполните его:
```bash
cp .env.example .env
```

Откройте `.env` и укажите:
- `DISCORD_TOKEN` - Токен вашего Discord бота
- `DATABASE_URL` - URL подключения к PostgreSQL

Пример:
```
DISCORD_TOKEN=your_discord_bot_token_here
DATABASE_URL=postgresql://user:password@localhost:5432/economy_db
```

### 4. Настройка PostgreSQL
Создайте базу данных:
```sql
CREATE DATABASE economy_db;
```

### 5. Запуск бота
```bash
python main.py
```

## 🔧 Создание Discord бота

1. Перейдите на [Discord Developer Portal](https://discord.com/developers/applications)
2. Нажмите "New Application" и введите имя
3. Перейдите в раздел "Bot" и нажмите "Add Bot"
4. Скопируйте токен бота в файл `.env`
5. В разделе "OAuth2" -> "URL Generator" выберите scopes: `bot`, `applications.commands`
6. Выберите разрешения: `Administrator` (или минимально необходимые)
7. Используйте сгенерированную ссылку для приглашения бота на сервер

## 📝 Примечания

- Бот использует PostgreSQL для хранения данных
- Все команды являются slash-командами (/)
- Базы данных создаются автоматически при первом запуске
- Ежедневная награда доступна раз в 24 часа
- У каждой профессии есть свой cooldown перед следующей работой

## 🛠 Технологии

- **Python 3.8+**
- **discord.py** - Библиотека для работы с Discord API
- **asyncpg** - Асинхронный драйвер PostgreSQL
- **PostgreSQL** - База данных
